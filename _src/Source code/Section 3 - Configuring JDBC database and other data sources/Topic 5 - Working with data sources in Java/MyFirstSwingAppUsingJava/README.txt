Installation instructions and description of the tasks.

------------------------------------------------------------------------
1. Prerequisites.
------------------------------------------------------------------------

Below the prerequisites for the installation.
- Eclipse installed.
To know how to prepare the environment, please see the video tutorial (Section 2, Topic 1).

------------------------------------------------------------------------
2. Preparing the MyFirstSwingAppUsingJava project.
------------------------------------------------------------------------

To import the MyFirstSwingAppUsingJava project, please the tasks described below:
- Open Eclipse -> File -> Import.
- General -> Existing projects into workspace -> Next.
- Complete the 'Select archive file' with the MyFirstSwingAppUsingJava.zip file -> Finish.

The project supposes to be stored in the '/home/pentaho/workspace' folder.
If this is not the case, please execute the tasks described below.

The first task is to update the 'providerUrl' variable in the MyFirstSwingAppUsingJava1 class.

The second task is to update the 'xmlDataFilePath' variable in the MyFirstSwingAppUsingJava2 class.

------------------------------------------------------------------------
3. Tasks and instructions.
------------------------------------------------------------------------

Select MyFirstSwingAppUsingJava1 class from Eclipse and run the application using the green button with the white arrow in the Run menu (or press Ctrl + F11).
Pressing the unique button in the new window with the 'Press me!' label, you will see the report.

For the second task, simply select MyFirstSwingAppUsingJava2 class from Eclipse and run the application using the green button with the white arrow in the Run menu (or press Ctrl + F11).
Pressing the unique button in the new window with the 'Press me!' label, you will see the report.
